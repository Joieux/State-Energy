# State-Energy

State-Energy is a simple static website that provides information and tools related to state energy (site content, forms, and interactive scripts). This repository contains the HTML, CSS, JavaScript, and assets for the site.

## Repository structure

- banner.png — hero/banner image used on the site
- index.html — main landing page
- homes.html — content page (homes/energy info)
- hear.html — content page
- form.html — form for user input
- script.js — JavaScript for site interactivity
- styles.css — global styles

## Local development

To preview the site locally (simple static site):

1. Clone the repository:

   git clone https://github.com/Joieux/State-Energy.git
   cd State-Energy

2. Open the files directly in your browser (double-click `index.html`) or run a local HTTP server:

   - With Python 3:
     python -m http.server 8000
     Open http://localhost:8000 in your browser

   - With Node.js (serve):
     npm install -g serve
     serve .

## Deployment

This repository can be hosted via GitHub Pages. In your repository settings, enable GitHub Pages and serve from the `main` branch (or `gh-pages` if you prefer a separate branch).
